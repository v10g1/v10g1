# My current focus
##  Smart Contract Security & Auditing

I specialize in identifying vulnerabilities and securing smart contracts to prevent exploits and safeguard user funds.

- **Security Best Practices:** Reentrancy protection, access control, gas optimization, and other industry-standard techniques.
- **Formal Verification & Static Analysis:** Experienced with tools like `Slither`, `Mythril`, and `Scribble` for automated vulnerability detection.
- **Manual Reviews:** Deep dives into Solidity codebases to uncover logic flaws, privilege escalations, and hidden attack vectors.
- **Security Research:** Constantly learning from recent exploits and emerging attack patterns to evolve my security strategies.
